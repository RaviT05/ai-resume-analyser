# AI Resume Analyser 🚀

An AI-powered web app that analyses your resume and gives detailed feedback instantly.

## What it does
- Upload your resume as a PDF
- AI reads and understands your resume
- Gives structured feedback including strengths, weaknesses, skill gaps and overall score

## Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3.3)
- PyPDF2

## How to run locally

1. Clone this repo
git clone https://github.com/RaviT05/ai-resume-analyser.git

2. Install dependencies
pip install streamlit groq pypdf2 python-dotenv

3. Create a `.env` file and add your Groq API key
GROQ_API_KEY=your-key-here

4. Run the app
python -m streamlit run app.py
